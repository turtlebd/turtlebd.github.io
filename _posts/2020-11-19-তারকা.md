## আমরা এখন turtle block কোডের মাধ্যমে উপরের তারকাটি অঙ্কন করব। 

#### নিচে turtle block কোড দেয়া হল।

![block code](https://i.ibb.co/TPJPGXS/figura-4.jpg)

#### নিচে python কোড দেয়া হল।

```python
import turtle

turtle.left (90)

for i in range (5):
    turtle.forward (150)
    turtle.right (144) 
```
