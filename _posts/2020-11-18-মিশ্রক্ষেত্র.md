## আমরা এখন turtle block কোডের মাধ্যমে উপরের মিশ্রক্ষেত্রটি অঙ্কন করব। 

#### নিচে turtle block কোড দেয়া হল।

![block code](https://i.ibb.co/TPJPGXS/figura-4.jpg)

#### নিচে python কোড দেয়া হল।

```python
import turtle

turtle. left (90)

for i in range (4):
    turtle.forward (50)
    turtle.left (90)
    turtle.forward (50)
    turtle.right (90)
    turtle.forward (50)
    turtle.right (90)    
```
